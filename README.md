# Homebrew Tap for Tameo

[Tameo](https://tameo.ati-mirai.co.jp) — an open-source (MIT) clipboard manager for Apple Silicon, a successor to Clipy.

## Install

```sh
brew install --cask supergodak/tap/tameo
```

Or:

```sh
brew tap supergodak/tap
brew install --cask tameo
```

- Site: https://tameo.ati-mirai.co.jp
- App repo: https://github.com/supergodak/tameo
